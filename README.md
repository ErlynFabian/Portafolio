# Portafolio Personal - Erlyn Valerio Fabian

## Descripción
Este es mi portafolio digital, diseñado para presentar mi trayectoria profesional, habilidades técnicas y proyectos como **Desarrollador Web Junior**. El objetivo principal es ofrecer una experiencia visual moderna, limpia y funcional que refleje mi compromiso con el código de calidad y el diseño centrado en el usuario.

## Características Principales
*   🌍 **Internacionalización (i18n)**: Soporte completo para dos idiomas (Español e Inglés), permitiendo cambiar el contenido de forma dinámica.
*   ⚡ **Rendimiento Optimizado**: Desarrollado con **Vite** y **React** para garantizar tiempos de carga ultrarrápidos.
*   ✨ **Experiencia Interactiva**: Navegación fluida y micro-animaciones implementadas con **Framer Motion**.
*   📱 **Diseño Responsivo**: Experiencia de usuario optimizada para cualquier dispositivo, desde monitores de escritorio hasta smartphones.
*   🎨 **Estética Moderna**: Interfaz minimalista con modo oscuro, tipografía cuidada y un sistema de diseño basado en **Tailwind CSS**.

## Secciones del Sitio
*   **Inicio**: Presentación rápida y contacto directo.
*   **Sobre mí**: Un resumen detallado de mi perfil profesional y aspiraciones.
*   **Experiencia**: Historial laboral detallado con tareas y logros clave.
*   **Habilidades**: Listado interactivo de competencias técnicas, habilidades blandas y certificaciones.
*   **Educación**: Formación académica relevante.
*   **Contacto**: Canales de comunicación integrados.
